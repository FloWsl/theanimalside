## 🚀 **Quick Start**

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Navigate to project directory
cd project

# Install dependencies
npm install

# Start development server
npm run dev
```

Your application will be available at `http://localhost:5173`
